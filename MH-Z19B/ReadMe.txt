Verwendetes Board : Arduino Mega 

Angeschlossene Geräte : 
	1. HC05 Modul
	2. MHZ19B

Verwendete Pins: 
	1. 3,3V für Stromzufuhr des HC-05
	2. 5V Für Stromzufuhr des MHZ19b
	3. GND für HC-05
	4. GND MHZ19b
	5. Pin 3 als Digitaler PWM Pin für MHZ19b Signalübertragung
	6. Pin 0 RX zu TX des HC-05
	7. Pin 1 TX zu RX des HC-05     
 
Beim Hochladen des Skripts auf das Board unbedingt pin 0 & 1 vom Board lösen. Ansonsten kommt es zum STK500 Error  oder Error: AVR Dude

Dieses Skript benötigt keine zusätzlichen Bibliotheksinstallationen.

Baudrate 9600 zwingend erforderlich!!

Range (Messbereich) ist im Skript anzupassen.
